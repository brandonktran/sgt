# sgt

This is a dynamic web application for teachers who want to record the grades of their students.


## Live Site
Try the live site at: http://sgt.brandonktran.com/

<img src="server/public/gifs/desktop.gif" style="display:inline-block;" width="98%">

## Technologies Used
React.js </br>
Node.js </br>
json-server </br>
Babel </br>
Webpack </br>
Bootstrap </br>
HTML </br>
CSS

## Main Features
1. User can add a grade </br>
2. User can delete a grade </br>
3. User can edit a grade </br>
4. User can view grades </br>


## Development
### System Requirements
Node.js 10 or higher <br>
NPM 6 or higher <br>

### Getting Started
Clone the repository.
```console
git clone https://github.com/brandonktran/sgt.git
cd sgt
```

Install all dependencies with NPM.
```console
npm install
```

Start the project.
```console
npm run dev
```
Then view the application by going to http://localhost:3000 in your browser.
